## Tips for This Exam

1. **Read scenarios carefully** - Many questions test understanding of specific configurations
2. **Eliminate wrong answers** - Usually 2 options can be quickly eliminated
3. **Watch for "most" and "best"** - These indicate a preferred approach among valid options
4. **Time management** - Average 2 minutes per question, flag difficult ones for review
5. **Compare with Mock Exam 1** - Track improvement across attempts
