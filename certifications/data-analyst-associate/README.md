# Databricks Data Analyst Associate

## Exam Overview

| Detail             | Information                                 |
| ------------------ | ------------------------------------------- |
| **Certification**  | Databricks Certified Data Analyst Associate |
| **Questions**      | ~45 multiple-choice                         |
| **Duration**       | 90 minutes                                  |
| **Passing Score**  | 70%                                         |
| **Languages**      | SQL                                         |
| **Experience**     | 6+ months with Databricks SQL               |
| **Recertification**| Every 2 years                               |
| **Cost**           | $200 USD                                    |
