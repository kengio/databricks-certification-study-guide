# Databricks Generative AI Engineer Associate

## Exam Overview

| Detail             | Information                                           |
| ------------------ | ----------------------------------------------------- |
| **Certification**  | Databricks Certified Generative AI Engineer Associate |

| **Passing Score**  | 70%                                                   |
| **Languages**      | Python                                                |
