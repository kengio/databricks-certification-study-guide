# Databricks Machine Learning Associate

## Exam Overview

| Detail             | Information                                     |
| ------------------ | ----------------------------------------------- |
| **Certification**  | Databricks Certified Machine Learning Associate |
| **Questions**      | ~45 multiple-choice                             |
| **Duration**       | 90 minutes                                      |
| **Passing Score**  | 70%                                             |
| **Languages**      | Python                                          |
| **Experience**     | 6+ months with Databricks ML                    |
| **Recertification**| Every 2 years                                   |
| **Cost**           | $200 USD                                        |
