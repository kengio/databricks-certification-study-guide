## Exam Overview

| Detail | Information |
|--------|-------------|
| **Certification** | Databricks Certified Data Engineer Associate |
| **Questions** | ~45 multiple-choice |
| **Duration** | 90 minutes |
| **Passing Score** | 70% |

| **Experience** | 6+ months with Databricks |
