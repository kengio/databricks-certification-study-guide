3. **Simulate**: Take full [Mock Exams](mock-exam/) under timed conditions
4. **Review**: Study areas where you struggled using topic README files

## Exam Format

- **Questions**: 45 multiple-choice
- **Duration**: 90 minutes
- **Passing Score**: 70%
- **Languages**: Python and SQL
