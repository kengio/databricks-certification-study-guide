# Required Screenshots: Delta Live Tables

This folder is intended to hold screenshots related to Lakeflow Declarative Pipelines (Delta Live Tables).

Please add the following screenshots and name them appropriately:

- `dlt_pipeline_graph.png`: The core DLT pipeline DAG view showing data flowing from Bronze to Silver to Gold.
- `dlt_data_quality_metrics.png`: A zoomed-in view of a table in the DLT graph showing row counts and expectation pass/fail/drop metrics.
- `dlt_pipeline_settings.png`: The configuration UI for a DLT pipeline, highlighting target schema, development mode, and continuous/triggered settings.
- `dlt_cluster_autoscale.png`: Cluster settings specifically tailored for a DLT pipeline showing enhanced autoscaling configuration.

Once these images are added, they can be referenced in the `07-lakeflow-pipelines` declarative pipeline markdown files.
